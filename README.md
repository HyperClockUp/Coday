## ⬇️ 下载

前往 **[Releases →](../../releases/latest)** 获取最新安装包：

| 平台 | 安装包 |
|------|--------|
| **Windows** | `Coday_x.x.x_x64-setup.exe` |
| macOS | 即将支持 |

## 安装

1. 下载 `Coday_x.x.x_x64-setup.exe`；
2. 双击运行，按提示完成安装；
3. 启动后按界面引导完成初始化即可。

> 安装时如出现一次系统授权（UAC）提示，点「是」允许即可。

## 🔄 自动更新

Coday 内置自动更新——有新版本时会在应用内提示，一键下载并升级，无需手动重新安装。

## 系统要求

- Windows 10 / 11（64 位）
- macOS 支持开发中
